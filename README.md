# 📘 Projet : Administration Système & Réseaux

##  Objectif

Ce projet regroupe plusieurs travaux pratiques réalisés dans le cadre de l’apprentissage de l’administration système, du réseau et du cloud.

---

##  Contenu du projet

###  Scripts Bash

Scripts permettant d’automatiser certaines tâches Linux :

* Script de bienvenue utilisateur
* Script de génération de rapport système

---

###  Scripts Python

* Fonction AWS Lambda en Python
* Traitement d’événements JSON

---

###  Linux / Terminal

Réponses aux exercices liés :

* Commandes de base
* Gestion des fichiers
* Surveillance système

---

###  Réseau (Packet Tracer)

Fichiers de simulation réseau contenant :

* Configuration d’un serveur HTTP
* Tests de connectivité (ping, HTTP)
* Topologies simples (LAN)

---

---

### 🖼️ Captures d’écran

Illustrations des résultats :

* Exécution des scripts
* Configuration réseau

---

## Concepts abordés

* Triade CIA (Confidentialité, Intégrité, Disponibilité)
* Sécurité des systèmes
* Réseaux informatiques (LAN, HTTP)
* Automatisation avec Bash

---

##  Comment utiliser ce projet

1. Cloner le dépôt :

```bash
git clone <lien-du-repo>
```

2. Accéder au dossier :

```bash
cd nom-du-projet
```

3. Exécuter les scripts Bash :

```bash
chmod +x script.sh
./script.sh
```
